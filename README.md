# Digital Business Card

A digital business card built with HTML and CSS — a clean, 
single-view profile card with contact info and social links. 
Built as a practice project using placeholder content and images.

Part of my journey learning full-stack development — an early 
project for practicing layout, styling, and visual detail.

## Live demo

[View it live](https://johnnydev4.github.io/business-card/)

## What it does

- Displays name, role, and a short bio
- Links to email and social profiles
- Clean card layout

## Built with

- HTML
- CSS

## What I learned

This project was about getting the details right on a small, 
focused layout — spacing, alignment, typography, and buttons. 
I practiced structuring a card component with HTML and styling it 
so it looks polished, paying attention to the visual hierarchy 
that makes a simple design feel intentional.
